# Division Andina Landing Page

## Descripción

Esta es una **landing page provisional** para la comunidad **Division Andina**, enfocada en *Arma 3* y juego táctico (*MILSIM*). Su objetivo es servir como punto de acceso para nuevos jugadores y visitantes.

## Qué muestra

- Enlaces a Discord y otros medios de contacto.

## Objetivo

Informar y guiar a los visitantes sobre cómo unirse a la comunidad y acceder a los servidores.

- 🌐 Sitio oficial: [divisionandinagaming.cl](https://www.divisionandinagaming.cl/)
- 🎮 Discord: disponible desde la landing page
